## Public Branch Policy

- Only public, traceable, author-created documentation relevant to the released artifact belongs here.
- Third-party bundles, binary attachments, future-planning files, local drafts, and mirrored upstream documentation are excluded from this snapshot.
- Updates to this directory must stay aligned with the shipped software baseline, the public research sources, and the reproducibility material.
